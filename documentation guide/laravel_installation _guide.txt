create project using composer or direct

install composerin root folder
composer install

and change database name in .env.example file



change .env.example to .env and add database name to .env file

then apply php artisan key:generate to generate key in .env file
after apply php artisan migrate to migrate table to database if any

then apply php artisan serve command
